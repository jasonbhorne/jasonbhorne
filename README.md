# Jason B. Horne, Ed.D., MBA, SFO

**Education Operations Leader | Data Systems | AI Integration**

I run the business side of a public school district and build the data systems that make it work. As Assistant Director of Schools for Administration at [Greeneville City Schools](https://www.gcschools.net/) in Tennessee, I oversee HR, finance, facilities, technology, transportation, nutrition, and data operations. I also teach graduate courses in educational leadership at [East Tennessee State University](https://www.etsu.edu/).

My background is in data science, and I write code to solve real operational problems: automating enrollment pipelines, digitizing paper-based reporting, analyzing school performance data, and building dashboards that help leaders make better decisions.

## What I Build

**Data Pipelines and Analytics**
Automated daily enrollment workflows that pull from PowerSchool, process through Python, and push to dashboards. Machine learning analysis of school performance metrics. Assessment data tools for Tennessee's TCAP/EOC frameworks.

**Operations Automation**
Task orchestration with dependency graphs, retry logic, and watchdog timers. File management systems that sort, categorize, and route documents across a district. Shell and Python scripts that replace hours of manual work.

**Reporting and Dashboards**
OCR-based digitization of paper forms into structured data. Automated Excel dashboard generation with year-over-year comparisons. Google Analytics reporting for web properties.
